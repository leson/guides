## install the USB driver manually

### download the archive package from [baidu pan](https://pan.baidu.com/s/1pLcZsnD
)

### decompress archive package 
```bash
tar -jxvf 0004-r8152.53-2.06.0.bz2
```
### change directory 
```bash
cd r8152-2.06.0
```

### compile/link/install
```bash
make 

sudo make install

```
